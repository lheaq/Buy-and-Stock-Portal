- Buy and Stock Portal is a Java Swing desktop application that simulates a basic school supply store system with two distinct user roles. Owners can log in with a password to either view current stock levels or add inventory across four product categories — pens, pad papers, notebooks, and erasers. Customers, on the other hand, can browse those same categories, add multiple items to a cart with specified quantities, and receive a summarized receipt showing itemized costs and the total payment. The entire interaction is handled through JOptionPane dialog boxes, with product data and pricing managed through a separate ProductStorage class.
- This is also a linked list mini program project for accessing products either as a customer who will order or an admin who will check or add stocks. This was made without prior knowledge to proper backend structure that is why it uses hardcoded backend and is not fixed.
